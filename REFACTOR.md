---

### 2. GUIDE.md - 删除 ❌

**问题：**
- 821 行详细指南
- 与 QUICKSTART.md 大量重复
- 有了 deploy.sh 一键部署后，详细步骤不再必要

**重复内容：**
```
GUIDE.md:
- 第一步：创建 mihomo VM (详细 Proxmox 步骤)
- 第二步：安装 Debian 12 (完整安装过程)
- 第三步：配置静态 IP (详细网络配置)
- 第四步：安装 mihomo
- 第五步：创建 AdGuard Home VM
- 第六步：配置 RouterOS
- 故障排查...

QUICKSTART.md:
- 准备工作 (创建 VM)
- 配置静态 IP
- 一键部署 (bash deploy.sh)
- 完成配置
- 测试验证
```

**对比：**
- GUIDE.md: 手动操作 30+ 步，60 分钟
- QUICKSTART.md + deploy.sh: 自动化，10 分钟

**解决：**
- 删除 GUIDE.md
- QUICKSTART.md 已足够
- 一键部署脚本替代了手动步骤
